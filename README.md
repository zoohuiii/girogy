# 가족 건강 기록 앱 MVP

가족 구성원의 건강 기록을 관리하는 간단한 웹 애플리케이션입니다.

## 기능

- 가족 구성원 선택
- 달력으로 날짜 확인
- 날짜별 건강 기록 작성 및 조회

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173`을 열어 확인하세요.

### 빌드

```bash
npm run build
```

## 기술 스택

- React 18
- React Router DOM
- Vite
